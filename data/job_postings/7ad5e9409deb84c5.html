Intern – Telco and Cloud Edge – Wireless Network Telemetry with ML/AI

Business Summary

Work Here. Transform Everywhere.

The newly acquired Uhana team at VMware harnesses the power of AI/ML and low latency big data pipelines to manage and optimize applications and mobile networks. Uhana provides cloud-based APIs that leverage real-time telemetry from mobile networks and applications and deep-learning to optimize application QoE and mobile network operations. Our Network AI engine ingests fine-grained wireless network telemetry from a variety of data sources, including the radio access network and core network and then processes the telemetry to optimize user experience.
