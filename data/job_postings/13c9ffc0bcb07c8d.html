Who You’ll Be Working With:

You’ll join a team of data scientists, machine learning engineers, and cloud engineers working to push the boundaries of data science and Artificial Intelligence (AI) in health care. We like to experiment, iterate, and innovate with technology, from developing new algorithms specific to health care’s challenges, to bringing the latest data science, machine learning, big data and cloud best practices and applications into the health care world. We know that Data Science and Artificial Intelligence are only valuable when powered by the right data, so we focus on fully gathering the data with proper data governance to understanding the problems we need to solve, and truly understanding the data behind them before launching into solutions – ensuring that the solutions we do land on are impactful and powerful.

What You’ll Be Doing:

  • Be the go-to person for Data ingest and storage across Cloud, Hadoop and Network storage devices
  • Support Data Scientists with ETL, Data Exploration, Feature Engineering, Model development and Model Deployment when required
  • Collaborate with stakeholders from clinical, administrative, operations, and actuarial teams to identify Data Sources, Data feeds and external third-party data for advanced analytics opportunities to add value to Evolent’s business
  • Build scalable data platforms across clouds and/or on premise as an alternate source of truth (ASOT) mirroring single source of truth (SSOT) data sources like RDBMS, Data Warehouse, SAS GRID, NoSQL and other external data to support Data Scientists and Machine Learning Engineers
  • Make Data available for Data Exploration, Data Science and Machine Learning purpose
  • Solution right storage platforms from object storage on AWS/AZURE/GCP, Hadoop Data lakes to on premise network drives with proper file formats, compression, partitions, backup and archive solutions
  • Create data catalog for data discovery and create reports for Data governance and ETL jobs
  • Ingest data batch and real time from Data warehouse, Data base, SAS data grid, MongoDB, NoSQL systems, SFTP and real time services on to Data Lake, S3 objects etc
  • Schedule jobs to ensure 24/7 availability, failover and graceful recovery for Data and ETL jobs
