Senior Software Engineer_IDM - Huntsville, AL

Job Description

Description

SAIC has an immediate need for a Senior Software Engineer to support the Systems Simulation, Software and Integration Directorate (S3I) in continuation of the Improved Data Modem (IDM) sustainment and development.
A Senior Software Engineer is needed to support complete software lifecycle development and sustainment activities for the AMCS Operational Environment (OE).

The IDM is a communication device for the Army rotorcraft digital fleet. It is an Airworthy hardware and software system. The S3I is supporting software sustainment for fielded IDM systems and an update to transition to the IDM 401 Multi-Core Processor (MCP). A Senior Software Engineer is needed to support complete software lifecycle development and sustainment activities for the IDM 401 MCP.

Duties and Responsibilities:
  • Support complete software development lifecycle for development
  • Perform Systems and Software Integration
  • Perform Unit Test and Code Coverage analysis
  • Produce software documentation (SDD, SVD, SUM,STP, STD ) sufficient to support airworthiness certification
  • Analyze system, software, and hardware specifications and interface documents to support transition of IDM to AMCS
  • Assist in the architecture, design and modeling of the IDM software
  • Provide leadership for interdisciplinary teams as required for design, documentation, implementation and test of incremental functional capabilities
