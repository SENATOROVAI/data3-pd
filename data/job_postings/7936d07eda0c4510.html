Software Engineer/ Computer Scientist - Dallas, TX

ASEC's current need is for a Software Engineer/ Computer Scientist/ with demonstrated experience in DoD R&D and fleet support of complex software-intensive naval weapons systems.

Education and Experience:

  • Bachelor's degree in Computer Science, Computer Engineering, Software Engineering or Engineering or demonstrated comparable experience with ten or more years of relevant experience

Required Clearance:

  • This position is subject to a government security investigation and must meet eligibility requirements for access to classified information at the Top Secret Must have a current Top Secret clearance

Skills and Qualifications:

  • Demonstrated understanding of real-time, embedded multi-computer systems in the areas of software development, documentation, configuration control, verification and validation is desired
  • Experience in software engineering activities e.g. requirements, design, integration, and test associated with developing mission critical computer software, integrating software intensive systems, and establishing and sustaining the capability to provide software life cycle support is desired
  • Demonstrated experience in analysis and design of real time software applications on large-scale, complex computer systems in airborne surveillance systems is desired
  • Experience in analysis and design of real time firmware applications on multiple FPGAs is desired

Tasks/Responsibilities:

  • Directly support the Software and Algorithm Support Activity as a team member
  • Provide horizontal integration of System Software across and within segments
  • Coordinate with Navy Sponsors and contractors to update/correct requirements/designs
  • Support Product and Functional IPT leads and IPT-embedded Software SMEs
  • Support Software Sustainment Lead in preparation of sustainment efforts
  • Support Statement of Work SOW development
  • Support Proposal evaluations
  • Support Engineering Data Requirements Agreement Plan EDRAP development
  • Support Schedule Risk Assessment SRA
  • Monitoring of testing at contractor site System Level II and System Level III testing
  • Consult with customers on status or technical issues, such as software system requirement, design, testing or maintenance
  • Represent the customer in status meetings with contractors
  • Consult with engineering staff to evaluate interface between hardware and software, develop specifications and performance requirements, or resolve customer problems
  • Review software documentation to ensure technical accuracy, traceability, and completeness. Documentation must fulfill the associated DID requirements. These documents include but not limited to:
    • Software Requirements Specifications
    • Software Design Document
    • Interface Design Document
    • Software Development Plans
    • Software Test Plans
    • Software Test Descriptions
    • Software Test Reports
    • Software Version Description SVD
  • Collaborate with engineers or software developers to select appropriate design solutions or ensure the compatibility of system concepts
  • Review system requirements and ensure flow down to appropriate functional configuration items
  • Participate in technical interchange meetings, and other events as required
  • Prepare reports or correspondence concerning technical reviews, specifications, activities, or status

Other Requirements:

  • Requires occasional travel
  • Requires flexibility in work hours
