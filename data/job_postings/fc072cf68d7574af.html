Data Science Intern - Sunnyvale, CA 94085

Job Description

As a data science intern at the Bosch Center for Artificial Intelligence you will have the opportunity to work on interesting machine learning problems related to autonomous driving, driver assistance technologies, sensor data analysis, spatio-temporal modeling and many more exciting topics. You will have access to state-of-the-art computing infrastructure and large industrial-scale data sets. You will also have the opportunity to improve products that impact millions of people worldwide.

Qualifications

Required Qualification:

  • Currently pursuing BS or MS or PhD in a related technical or quantitative field
  • Strong foundations in machine learning and data visualization
  • Proficient in programming in Python or R
  • Good software engineering skills. Able to write efficient, readable and tested code
  • Be enthusiastic and able to collaborate in a team environment
  • Ability to clearly and simply communicate technical concepts
  • Curious and passionate about learning everything

Desired Qualification:

  • Experience working with large data sets in a distributed computing environment (e.g. Apache Spark)
  • Visualization tools & techniques for large data sets
  • Active github that demonstrates past work
  • Domain knowledge of manufacturing and assembly line processes

Additional Information

  • Start Date: Spring 2020
  • Duration: 12 - 24 weeks
