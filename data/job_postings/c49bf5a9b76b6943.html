TriVector Services is seeking a Senior Mission Equipment Software Airworthiness Engineer with professional experience in software safety standards such as MIL-STD-882, Civil guidance documents SAE ARP 4754, SAE ARP 4761, Civil and DoD aviation software life-cycle artifacts. This position requires the incumbent to:

Description

  • Review Software requirements and associated documentation such as System/Subsystem Specification (SSS), Software Requirements Specification (SRS), Interface Requirement Specification (IRS), and requirements traceability matrices
  • Review Software program planning documentation such as Software Development Plan (SDP)
  • Review Software design and associated documentation such as System/Subsystem Design Description (SSDD), Software Design Description (SDD), and Interface Design Description (IDD)
  • Participate in Technical Interchange Meetings (TIM), Preliminary Design Reviews (PDR), Critical Design Reviews (CDR), and Test Readiness Reviews (TRR)
  • Review Software related issues in Safety documentation such as System Safety Assessment Reports, and System Safety Hazard Analysis (SSRA), Functional Hazard Assessment (FHA), and Failure Mode, Effects and Criticality Analysis (FMECA)
  • Review Software test documentation such as Test and Evaluation Master Plan (TEMP), Software Test Plan (STP), Software Test Description (STD), and Software Test Report (STR)
  • Provide test witnessing of software tests
  • Review Airworthiness documentation such as Airworthiness Qualification Plan (AQP), Airworthiness Qualification Specification (AQS), Plan for Software Aspects of Certification (PSAC), Software Accomplishment Summary (SAS), and Airworthiness Release (AWR)
  • Review Software version for release documentation such as the Software Version Description (SVD) and Software Product Specification (SPS)
  • Support Government in the development of Guidance, Processes, Practices, and Requirements for Software Development and Software Airworthiness Qualification

Skills and Requirements

  • RTCA DO-178, Software Considerations in Airborne Systems and Equipment Certification
  • Civil and DoD aviation software life-cycle artifacts
  • Army airworthiness process regulations AR 70-62 and AMCOM 385-17
  • DOD Data Item Descriptions (DiD)s
  • Software requirements traceability
  • IBM Rational DOORS

Security clearance: U.S. Citizenship is required for this position.
