Global Data Engineer - Boston, MA

  • Collaboration with the Data Center Subject Matter Experts, Data Scientists, and Program Managers
  • Create and manage large datasets by extracting, transforming, combining, and loading data from various heterogeneous data sources
  • Maintain data integrity, availability, and auditability
  • Identify gaps and improvement opportunities in existing data infrastructure
  • Design, implement, and maintain a cutting-edge cloud-based data-infrastructure for large data-sets
  • Build and Maintain serverless data ingestion and refresh pipelines in terabyte scale using Azure cloud services
  • Drive the adoption of new technologies and new best practices
  • Incorporate new data sources from external vendors using flat files, APIs, web-scraping, and databases
  • Able to match external data sources to existing data using exact and fuzzy methods
  • Maintain and provide support for the existing data pipelines using Python
  • Develop and implement tests to ensure data quality across all integrated data sources
  • Serve as internal subject matter expert and coach to train team members in the use of distributed computing frameworks for data analysis and modeling

Job Qualifications
  • Master’s degree in Computer Science, Engineering, or equivalent work experience
  • Two to four years’ experience working with datasets with hundreds of millions of rows using a variety of technologies
  • A deep knowledge of a variety of data engines like SQL Server and MySQL
  • Intermediate to expert level programming experience in Python and SQL in Windows or Mac/Linux environment
  • Experience analyzing data for data quality and supporting the use of data in an enterprise setting
  • Intermediate level experience working with distributed computing frameworks and with relational databases such as Microsoft SQL Server
  • Experience working with Databricks, Azure Data Warehouse and contemporary data file formats like Apache Parquet
  • Experience working with JavaScript, Azure DevOps, PowerBI. Terraform, clustering and classification models, a plus
